REQUIREMENTS
============

You need to install the Coins module first:
http://svn.csphere.eu/module/mindcrime/mods/coins.zip

You need to install the tools.php file to system/runstartup/:
http://svn.csphere.eu/module/mindcrime/general/system/runstartup/tools.php

INSTALL
=======

1) Upload all files (except in updates/) to your webserver
2) System -> Database -> Import the following files (in that order):
bets_install.sql
3) Clear the cache
