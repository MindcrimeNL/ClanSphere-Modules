
This folder contains general code, required for several modules

Twitter module requires:
- datacache module installed (http://www.csphere.eu/modules/module/view/id/231)
- system/runstartup/tools.php

TeamSpeak module requires:
- system/runstartup/tools.php

Bets module requires:
- system/runstartup/tools.php

Cups module requires:
- system/runstartup/tools.php

Replays module requires:
- system/runstartup/tools.php

IMPORTANT:
----------

If you have ClanSphere 2010.x or newer you need to put in your setup.php:

$cs_main['runstartup'] = TRUE;
